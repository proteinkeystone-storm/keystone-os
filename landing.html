Comment ça marche

3 étapes pour démarrer

1
Activez votre licence

Entrez votre clé au format XXXX-XXXX-XXXX-XXXX. Vos outils autorisés apparaissent instantanément dans le Dashboard.

2
Configurez vos moteurs

Ajoutez vos clés API (Claude, GPT-4o, Gemini…) dans le Coffre-fort chiffré. Choisissez le moteur par défaut pour chaque outil.

3
Générez & Exportez

Remplissez les champs, appuyez sur Générer. Obtenez une réponse structurée. Exportez en PDF ou copiez pour votre CRM.
